= 卸载 FlexVol 卷
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 中存储窗格的接合路径选项从 Storage Virtual Machine （ SVM ）命名空间中的接合卸载 FlexVol 卷。

.步骤
. 单击 * 存储 * > * 接合路径 * 。
. 从 * SVM* 字段的下拉菜单中，选择要从中卸载卷的 SVM 。
. 选择必须卸载的卷，然后单击 * 卸载 * 。
. 选中确认复选框，然后单击 * 卸载 * 。
