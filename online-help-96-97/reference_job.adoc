= 作业窗口
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用作业窗口管理作业任务，例如显示作业信息和监控作业进度。



== 命令按钮

* * 刷新 *
+
更新窗口中的信息。





== 选项卡

* * 当前作业 *
+
此选项卡显示有关正在进行的作业任务的信息。

* * 作业历史记录 *
+
此选项卡显示有关所有作业的信息。





== 作业列表

* * 作业 ID*
+
显示作业的 ID 。

* * 开始时间 *
+
显示作业的开始时间。

* * 作业名称 *
+
显示作业的名称。

* * 节点 *
+
显示节点的名称。

* * 状态 *
+
显示作业的状态。

* * 作业问题描述 *
+
显示作业的问题描述。

* * 进度 *
+
显示作业的状态。

* * 计划名称 *
+
显示计划的名称。
