= FC/FCoE 窗口
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 FC/FCoE 窗口启动或停止 FC 服务。



== 命令按钮

* * 编辑 * 。
+
打开编辑节点名称对话框，在此可以更改 FC 或 FCoE 节点名称。

* * 启动 *
+
启动 FC/FCoE 服务。

* * 停止 *
+
停止 FC/FCoE 服务。

* * 刷新 *
+
更新窗口中的信息。





== FC/FCoE 详细信息

详细信息区域显示有关 FC/FCoE 服务状态，节点名称和 FC/FCoE 适配器的信息。
