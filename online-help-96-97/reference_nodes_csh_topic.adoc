= 节点窗口
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用节点窗口查看集群中节点的详细信息。



== 命令按钮

* * 初始化 ComplianceClock*
+
将选定节点的 ComplianceClock 初始化为系统时钟的当前值。

* * 刷新 *
+
更新窗口中的信息。





== 节点列表

* * 名称 *
+
显示节点的名称。

* * 状态 *
+
显示节点的状态（节点是已启动还是已关闭）。

* * 运行时间 *
+
显示节点运行的持续时间。

* * ONTAP 版本 *
+
显示节点上安装的 ONTAP 版本。

* * 型号 *
+
显示节点的平台型号。

* * 系统 ID*
+
显示节点的 ID 。

* * 序列号 *
+
显示节点的序列号。





== 详细信息区域

显示有关选定节点的详细信息。

* * 详细信息选项卡 *
+
显示与选定节点相关的信息，例如节点名称，节点状态以及节点运行的持续时间。

* * 性能选项卡 *
+
显示选定节点的吞吐量， IOPS 和延迟。

+
更改客户端时区或集群时区会影响性能指标图。您应刷新浏览器以查看更新后的图形。
