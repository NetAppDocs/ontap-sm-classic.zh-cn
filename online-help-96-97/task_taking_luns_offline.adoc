= 使 LUN 脱机
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 中的 * LUN 管理 * 选项卡使选定的 LUN 脱机并使其无法用于块协议访问。

访问 LUN 的任何主机应用程序都必须暂停或同步。

.步骤
. 单击 * 存储 * > * LUNs * 。
. 在 * LUN 管理 * 选项卡中，选择一个或多个要脱机的 LUN 。
. 单击 * 状态 * > * 脱机 * 。
