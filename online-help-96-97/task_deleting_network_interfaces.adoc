= 删除网络接口
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 删除网络接口以释放该接口的 IP 地址，然后将该 IP 地址用于其他目的。

必须禁用网络接口的状态。

.步骤
. 单击 * 网络 * > * 网络接口 * 。
. 选择要删除的接口，然后单击 * 删除 * 。
. 选中确认复选框，然后单击 * 删除 * 。
