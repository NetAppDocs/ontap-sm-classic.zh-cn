= 恢复镜像关系
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以恢复已暂停的镜像关系。恢复此关系后，系统将恢复向镜像目标的正常数据传输，并重新启动所有镜像活动。

如果您已从命令行界面（ CLI ）暂停已断开的镜像关系，则无法从 System Manager 恢复此关系。您必须使用命令行界面恢复此关系。

.步骤
. 单击 * 保护 * > * 卷关系 * 。
. 选择要恢复的镜像关系。
. 单击 * 操作 * > * 恢复 * 。
. 选中确认复选框，然后单击 * 恢复 * 。


此时将恢复选定镜像关系中到镜像目标的数据传输。
