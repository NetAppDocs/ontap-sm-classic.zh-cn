= 更改集群密码
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 重置集群的密码。

.步骤
. 单击 * 配置 * > * 集群 * > * 配置更新 * 。
. 在 * 集群详细信息 * 窗格中，单击 * 更改密码 * 。
. 在 * 更改密码 * 对话框中，指定新密码，确认新密码，然后单击 * 更改 * 。
