= 正在删除计划
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 删除运行特定存储管理任务的计划。

.步骤
. 单击 * 保护 * > * 计划 * 。
. 选择要删除的计划并单击 * 删除 * 。
. 选中确认复选框，然后单击 * 删除 * 。
