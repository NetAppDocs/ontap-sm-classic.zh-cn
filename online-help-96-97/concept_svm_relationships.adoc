= SVM 关系
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage Virtual Machine （ SVM ）灾难恢复（ DR ）可通过恢复 SVM 成分卷中的数据以及恢复 SVM 配置，在 SVM 级别提供灾难恢复功能。

您可以使用 System Manager 创建和管理 SVM 之间的镜像关系以及镜像和存储关系。
