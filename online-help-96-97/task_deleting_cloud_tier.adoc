= 删除云层
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 删除不再需要的云层。

您必须已删除与云层关联且已启用 FabricPool 的聚合。

.步骤
. 单击 * 存储 * > * 聚合和磁盘 * > * 云层 * 。
. 选择要删除的云层，然后单击 * 删除 * 。
