= 更改 SVM 用户帐户的密码
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 重置 Storage Virtual Machine （ SVM ）用户帐户的密码。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择 SVM ，然后单击 * SVM 设置 * 。
. 在 * SVM 用户详细信息 * 窗格中，单击 * 用户 * 。
. 选择要修改密码的用户帐户，然后单击 * 重置密码 * 。
. 在 * 重置密码 * 对话框中，键入新密码并确认新密码，然后单击 * 更改 * 。
