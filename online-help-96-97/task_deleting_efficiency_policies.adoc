= 正在删除效率策略
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Managerto 删除不再需要的效率策略。

必须禁用效率策略。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择 SVM ，然后单击 * SVM 设置 * 。
. 在 * 策略 * 窗格中，单击 * 效率策略 * 。
. 选择要删除的效率策略，然后单击 * 删除 * 。
. 选中确认复选框，然后单击 * 删除 * 。
