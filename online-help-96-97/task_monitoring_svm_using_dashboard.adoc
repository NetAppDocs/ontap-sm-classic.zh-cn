= 监控 SVM
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以通过 System Manager 中的信息板监控 Storage Virtual Machine （ SVM ）的运行状况和性能。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择要监控的 SVM 的名称。
. 在信息板面板中查看详细信息。
