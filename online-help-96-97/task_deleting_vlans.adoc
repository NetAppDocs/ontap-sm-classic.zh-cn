= 正在删除 VLAN
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 删除在网络端口上配置的 VLAN 。在从插槽中删除 NIC 之前，您可能需要删除 VLAN 。删除 VLAN 时，此 VLAN 将自动从使用此 VLAN 的所有故障转移规则和组中删除。

不能将任何 LIF 与 VLAN 关联。

.步骤
. 单击 * 网络 * > * 以太网端口 * 。
. 选择要删除的 VLAN ，然后单击 * 删除 * 。
. 选中确认复选框，然后单击 * 删除 * 。
