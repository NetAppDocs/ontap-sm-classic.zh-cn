= 什么是 FabricPool
:icons: font
:imagesdir: ../media/


[role="lead"]
FabricPool 是一种混合存储解决方案，使用全闪存（全 SSD ）聚合作为性能层，使用对象存储作为云层。FabricPool 中的数据会根据是否经常访问而存储在层中。使用 FabricPool 可以帮助您降低存储成本，而不会影响性能，效率或保护。
