= 启动或停止 iSCSI 服务
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 在存储系统上启动或停止 iSCSI 服务。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择 SVM ，然后单击 * SVM 设置 * 。
. 在 * 协议 * 窗格中，单击 * iSCSI * 。
. 根据需要单击 * 开始 * 或 * 停止 * 。
