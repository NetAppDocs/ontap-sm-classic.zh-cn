= 正在删除导出策略规则
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 删除不再需要的导出策略规则。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择 SVM ，然后单击 * SVM 设置 * 。
. 在 * 策略 * 窗格中，单击 * 导出策略 * 。
. 选择要删除导出规则的导出策略。
. 在 * 导出规则 * 选项卡中，选择要删除的导出规则，然后单击 * 删除 * 。
. 在确认框中，单击 * 删除 * 。
