= 正在启动 SVM
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 通过启动 Storage Virtual Machine （ SVM ）来提供数据访问。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择要启动的 SVM ，然后单击 * 启动 * 。


SVM 将开始向客户端提供数据。
