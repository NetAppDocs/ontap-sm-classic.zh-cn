= 编辑效率策略
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 System Manager 修改效率策略的属性，例如策略名称，计划名称和最长运行时间。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择 SVM ，然后单击 * SVM 设置 * 。
. 在 * 策略 * 窗格中，单击 * 效率策略 * 。
. 在 * 效率策略 * 窗口中，选择要编辑的策略，然后单击 * 编辑 * 。
. 在 * 编辑效率策略 * 对话框中，进行所需的更改。
. 单击 * 保存 * 。
