= DNS/DDNS 服务窗口
:icons: font
:imagesdir: ../media/


[role="lead"]
通过 DNS/DDNS 服务窗口，您可以查看和编辑系统的当前 DNS 和 DDNS 设置。



== 命令按钮

* * 编辑 * 。
+
打开编辑 DNS/DDNS 设置对话框，在此可以添加或修改 DNS 或 DDNS 详细信息。您还可以启用或禁用 DDNS 。

* * 刷新 *
+
更新窗口中的信息。
