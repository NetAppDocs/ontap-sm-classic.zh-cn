= 什么是滞后时间
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
滞后时间是指目标系统滞后于源系统的时间量。

滞后时间是当前时间与上次成功传输到目标系统的 Snapshot 副本的时间戳之间的差值。除非源系统和目标系统上的时钟未同步，否则滞后时间始终至少与上次成功传输的持续时间相同。时区差异会自动计算为滞后时间。
