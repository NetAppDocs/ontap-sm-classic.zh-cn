= 配置对等关系（ ONTAP 9.2 及更早版本）
:icons: font
:imagesdir: ../media/


[role="lead"]
对等关系用于定义网络连接，使集群和 SVM 能够安全地交换数据。您必须先创建集群对等关系，然后才能创建 SVM 对等关系。
