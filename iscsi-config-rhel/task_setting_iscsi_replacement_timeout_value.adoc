= 设置 iSCSI 替换超时值
:icons: font
:imagesdir: ../media/


[role="lead"]
要支持多路径，必须修改 iSCSI 配置文件中的替换超时值。

.步骤
. 将 ` /etc/iscsi/iscsid.conf` 文件中的替换超时值从默认值 120 更改为 5 。
+
替换超时值在 `iscsid.conf` 中类似于以下内容：

+
[listing]
----
node.session.timeo.replacement_timeout = 5
----
